## 选择标准
请根据以下标准选择最合适的词语：

1. **语义连贯**：与上下文语义自然衔接
2. **声调匹配**：符合旋律的声调要求（如有指定）
3. **韵律协调**：如处押韵位置，需考虑韵脚
4. **主题一致**：符合歌词整体主题和情感基调
5. **用词自然**：优先选择常用、自然的表达

## 输出格式
请以 JSON 格式输出你的选择：

```json
{{
    "word": "选中的词语",
    "reason": "详细的选择理由，解释为何该词最符合上述标准",
    "alternatives": ["备选词 1", "备选词 2", "备选词 3"],
    "confidence": 0.85
}}
```

其中：
- `word`: 必须是从候选列表中选择的一个词
- `reason`: 详细解释选择理由
- `alternatives`: 2-3 个备选词（如有）
- `confidence`: 0-1 之间的置信度分数
