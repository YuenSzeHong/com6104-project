# Cantonese Lyrics Agent

AI Agent for automatic Cantonese lyrics generation using LangChain + Ollama.

[English](README.md) | [中文版](README.zh-CN.md)

## Project Description

This project is an automatic Cantonese Lyrics filling system based on Agentic AI. The user inputs MIDI melody and lyrics/artistic text, and the AI ​​Agent automatically analyzes the melody syllables, understands the artistic conception of the original words, and generates new lyrics that comply with Cantonese pronunciation and rhyme rules
